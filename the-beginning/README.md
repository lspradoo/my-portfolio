# The Roots of My Journey

This folder contains the very first projects I built when I decided to merge my Marketing background with the power of software development. 

## How it started
Coming from a Marketing DNA, I always understood the "why" behind user behavior, but I felt limited by not knowing the "how" to build the solutions myself. These projects represent the exact moment I fell in love with the logic of code.

What started as a curiosity about how to make things "clickable" quickly turned into a passion for solving real problems. I realized that programming is the ultimate marketing tool: it allows you to build experiences that actually solve the user's pain points in real-time.

## What is inside
The projects below were my training ground for JavaScript logic, DOM manipulation, and CSS architecture:

* **Calculator:** My first deep dive into mathematical logic and state management in JS.
* **Interactive Quiz:** An experiment in user engagement and conditional rendering.
* **Task List:** A study on data persistence and CRUD fundamentals.

Today, I build professional Chrome Extensions and corporate platforms, but I keep these projects here as a reminder of the discipline and curiosity that drive my career every day.
